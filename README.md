**오늘 내가 풀어야 할 알고리즘 문제, 데이터가 골라줍니다.**

## ✨ 핵심 기능

### 📅 풀이 캘린더

<!-- TODO: 캘린더 스크린샷 p1 추가 -->

월별 달력에서 나의 풀이 기록을 한눈에 확인할 수 있습니다.
풀 예정 문제를 미리 등록해두거나, 날짜별로 어떤 알고리즘 유형을 풀었는지 색상으로 구분해 볼 수 있습니다.

- 날짜별 풀이 기록(solved) 및 예약(will-solve) 문제 표시
- 드래그 앤 드롭으로 풀이 순서 변경
- 미니 캘린더 사이드바로 빠른 날짜 이동

---

### 🎯 개인화 문제 추천

내 풀이 이력을 분석해 오늘 풀기에 최적인 문제를 추천합니다.
난이도와 알고리즘 유형을 직접 선택해 원하는 방향으로 학습할 수 있습니다.

- 난이도 필터 (Easy / Normal / Hard / Extreme)
- 60여 개 알고리즘 유형 중 직접 선택
- 문제별 추천 이유 제공
- 마음에 드는 문제를 캘린더에 바로 등록

---

### 📊 태그별 숙련도 대시보드

<!-- TODO: 태그 대시보드 스크린샷 p2 추가 -->

내가 어떤 알고리즘 유형을 잘하고, 어디가 부족한지 한눈에 파악할 수 있습니다.
60여 개 알고리즘 태그마다 INTERMEDIATE → ADVANCED → MASTER 3단계 숙련도를 추적합니다.

- 태그별 진도 바 및 등급 표시
- 이름순, 마지막 풀이일, 레벨, 진도 정렬
- 태그 검색 및 카테고리 필터

---

### 👥 스터디 그룹

<!-- TODO: 스터디 스크린샷 p3 추가 -->

혼자가 아닌 함께 공부하고 싶다면 스터디 그룹을 만들어 보세요.
팀원을 초대하고, 팀 기반 문제 추천을 받아 함께 학습할 수 있습니다.

- 스터디 생성 및 멤버 초대/신청
- 팀원 공동 캘린더 및 문제 할당
- 팀 기반 문제 추천 및 추천 이력 관리
